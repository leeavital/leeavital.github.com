Ethereal Commerce


On Github

Ethereal Commerce is the first large scale PHP framnework I every developed. It’s a framework for creating themable ecommerce system with a admin dashboard. It’s based heavily on (older versions) of wordpress — in other words, there are global functions everywhere!

When I first got into PHP I was really into wordpress, so I wanted to build a themable system just like wordpress.

Take a look at a basic template file:

<?php get_header(); ?>

<?php while ($epc_query->hasNext()): ?>


<a href="<?= p_href(); ?>">
<?php p_title(); ?> <br />
</a>
<?php p_description(); ?> <br />
<?php p_controls(); ?> <br />

<?php p_category(); ?>

<br />
<br />
<br />

<?php $epc_query->next(); ?>
<?php endwhile ?>

<?php get_footer(); ?>
